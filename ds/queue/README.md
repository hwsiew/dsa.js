# Queue
Queue follows the First In First Out (FIFO) rule - the item that goes in first is the item that comes out first, except priority queue.

## Types of Queue
1. Simple Queue
2. Circular Queue
3. Priority Queue
4. Double Ended Queue

## Complexity Analysis 
       | Simple | Circular | Priority | Double Ended 
-------|--------|----------|----------|--------------
Enqueue|O(1)|O(1)| |
Dequeue|O(1)|O(1)| |
