Class Heap

Heap

A Heap is a special Tree-based data structure in which the tree is a complete binary tree

Examples

 
const { Heap } = require('dsajs');
// or  
// import { Heap } from 'dsajs';
let maxHeap = new Heap((a,b) => b>a);
let minHeap = new Heap((a,b) => a>b);
